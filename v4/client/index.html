Vision System

Advanced Object Recognition Platform

System Overview

The Vision System is a comprehensive object recognition platform that leverages cutting-edge machine learning models to detect and identify objects in real-time camera streams and static images. Built with MediaPipe's EfficientDet Lite2 model, the system provides accurate, fast, and configurable object detection capabilities.

The platform is designed with a modular architecture, allowing for easy configuration of detection classes, thresholds, and custom action functions. It supports both real-time processing from camera feeds and batch processing of uploaded images, making it suitable for a wide range of applications including security monitoring, content analysis, and automated object tracking.

Key Features

  • Real-time Object Detection: Process live camera feeds with configurable recognition intervals
  • Batch Image Processing: Upload images from files or URLs for analysis
  • Customizable Detection Classes: Configure which object types to detect (persons, vehicles, animals, etc.)
  • Bounding Box Visualization: Visual feedback with customizable bounding boxes and labels
  • Action System: Execute custom functions based on detection results
  • OpenAI Integration: Optional AI-powered image description capabilities
  • Modular Architecture: Easy to extend and customize for specific use cases

Camera Stream

Launch Camera Stream

Real-Time Object Recognition

The Camera Stream module provides live object detection from your device's camera feed. It continuously processes video frames and displays detected objects with bounding boxes overlaid on the video stream.

Capabilities:

  • Live Camera Feed: Access and display your device's camera stream as a background
  • Continuous Recognition: Automatically detect objects at configurable intervals (default: 500ms)
  • Manual Recognition: Trigger recognition on-demand for specific frames
  • Real-time Bounding Boxes: Visual overlay showing detected objects with confidence scores
  • Recognition Actions: Execute custom functions when objects are detected
  • Regular Actions: Periodic actions that run independently of recognition cycles

Use Cases:

  • Security monitoring and surveillance
  • People counting and tracking
  • Real-time object presence detection
  • Interactive applications requiring live visual feedback

Controls:

  • Start Camera: Initialize and display the camera stream
  • Stop Camera: Stop the camera feed and release resources
  • Start Recognition: Begin continuous object detection
  • Stop Recognition: Pause detection while keeping camera active
  • Manual Recognition: Perform a single recognition pass on the current frame

Image Upload

Launch Image Upload

Static Image Analysis

The Image Upload module allows you to analyze static images by uploading files from your device or loading images from URLs. It performs object detection and displays the results with bounding boxes, allowing you to download the annotated image.

Capabilities:

  • File Upload: Select and process images from your local device (JPG, PNG formats)
  • URL Loading: Load images directly from web URLs for remote analysis
  • Single-Pass Recognition: Analyze images on-demand with manual trigger
  • Result Visualization: Display images with bounding boxes and confidence labels
  • Image Download: Save annotated images with detection results
  • Manual Recognition Actions: Execute custom functions based on detection results

Use Cases:

  • Batch image analysis and processing
  • Content moderation and filtering
  • Image annotation and labeling
  • Quality control and object verification
  • Research and data collection

Workflow:

  1. Select an image file or enter an image URL
  2. Click "Recognize" to perform object detection
  3. View the results with bounding boxes overlaid
  4. Download the annotated image if needed

Technical Details

Model

MediaPipe EfficientDet Lite2 - A lightweight, efficient object detection model optimized for real-time performance

Configuration

Centralized configuration system allows customization of detection classes, thresholds, intervals, and action functions

Architecture

Modular design with separate client modules and shared library functions for recognition, actions, and visualization

API Integration

Optional OpenAI GPT-4 Vision API integration for advanced image description and analysis capabilities
